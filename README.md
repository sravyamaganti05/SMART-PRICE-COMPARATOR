
 Smart Price Comparator

A web application that helps users compare prices across multiple platforms to find the best deals instantly.

---
  Live Demo
> http://ec2-13-61-25-83.eu-north-1.compute.amazonaws.com/

---

 Tech Stack

- **Frontend:** HTML, JavaScript
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Package Manager:** npm / pnpm
- **Linting:** ESLint, Prettier

---

 Getting Started

### Prerequisites
- Node.js (v18 or higher)
- npm or pnpm

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/smart-price-comparator.git

# Navigate to project folder
cd smart-price-comparator

# Install dependencies
npm install

# Start development server
npm run dev
